# Admins

Admin role for the Warsaw AI Community platform. See [`governance.md`](./governance.md) for the role's authority and appointment process.

The platform's RBAC reads this file at build time to grant admin capabilities. In v0.1, admin is an RBAC marker only — UI-distinct admin features arrive in v0.2+ per [`projects/community-platform/spec.md` §4](../../projects/community-platform/spec.md).
