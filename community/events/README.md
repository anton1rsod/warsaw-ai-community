# Events

One folder per ad-hoc event: launches, news-driven bursts, guest speakers, hackathons.

## Convention

Folder name: `YYYY-MM-DD-slug/`

Each event folder contains:
- `README.md` — what it was, when, who showed up.
- `pitch.md` — the original pitch/proposal.
- `outcomes.md` — what came out of it (ideas, projects, decisions). `#kb` by default.
- optional: `artifacts/` — slides, screenshots, recordings (links or files).
