# Weekly Meetings

One markdown file per week. Copy [`_template.md`](_template.md) to `YYYY-MM-DD.md` (date of the meetup).

## Convention

- File name: `YYYY-MM-DD.md` (ISO date of the meetup).
- File is written by the host within 48h of the meetup.
- Gbrain ingests these by default (`#kb` implicit).
