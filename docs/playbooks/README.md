# Playbooks

Runbooks and how-tos for organizers and active members. Unlike ADRs (decisions) or specs (designs), playbooks describe **how to do repeatable tasks**.

## Current playbooks

- [GBrain Operations](gbrain-operations.md) — deploy, monitor, intervene, recover.
